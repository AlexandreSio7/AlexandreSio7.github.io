Création du Portfolio Professionnel

📋 Contexte du projet

Développement d'un support numérique pour présenter mes compétences et mes réalisations lors de l'épreuve E4 et E6 du BTS SIO. L'objectif était de créer une interface moderne, réactive (responsive) et hébergée de manière professionnelle.

🛠️ Réalisation Technique

  • Développement en HTML5 / CSS3 (Flexbox & Grid).
  • Utilisation de Git pour le versionnage du code.
  • Déploiement via GitHub Pages pour un hébergement gratuit et sécurisé (HTTPS).
  • Intégration d'un formulaire de contact via Formspree (API).

✅ Compétences Validées

B1.1 - Patrimoine informatique B1.3 - Mode Projet Web

Mode Projet : Organisation des tâches (Design > Développement > Tests > Mise en ligne). Gestion de la documentation technique et maintenance du dépôt GitHub.

🚀 Hébergement : Ce site est actuellement en ligne sur les serveurs de GitHub.
